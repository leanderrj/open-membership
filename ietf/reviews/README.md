# Reviews of `draft-om-rss-membership-00`

This directory holds the five external reviews of the Open Membership RSS Internet-Draft, conducted in Phase 5 Month 14. Reviews land here as pull requests; the protocol editor merges as-is and triages comments separately into [`../REVIEW-DISPOSITIONS.md`](../REVIEW-DISPOSITIONS.md).

Each review document follows the format defined in [`../REVIEW-PACKAGE.md`](../REVIEW-PACKAGE.md) §4.

## Filenames

- `review-implementer-1-<initials>.md`, independent CMS-side implementer
- `review-implementer-2-<initials>.md`, independent reader-side implementer
- `review-implementer-3-<initials>.md`, PSP integration engineer
- `review-security-<initials>.md`, security reviewer (non-implementer)
- `review-privacy-<initials>.md`, privacy reviewer (non-implementer)
