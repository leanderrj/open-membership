The protocol editor's record of how every comment from the M14 external review was triaged into the `-01` revision. Populated during M14 Week 4. Public from the moment the first entry lands.

For the review window, briefs, return format, and editorial process, see [`REVIEW-PACKAGE.md`](REVIEW-PACKAGE.md).

---

## How to read this file

Each entry is one comment from one review. Order is by reviewer, then by line number ascending. The disposition is one of:

- **accept**, landed in `-01` as a diff. Citation links to the commit.
- **decline**, did not land. Citation gives the reason; the reviewer is notified on the GitHub issue.
- **defer**, moved to the 1.x errata backlog. Citation links to the backlog issue.

Conflict between reviewers (one wants accept, another wants decline) is resolved by majority vote of the working group per [`../docs/GOVERNANCE.md`](../docs/GOVERNANCE.md) §"Decision-making." Both perspectives are recorded.
